Maven Plugin for Eclipse Virgo

This maven plugin provides goals to use some of the functions provided by the Virgo MBean server.

General

The project is published under the Eclipse Public License (EPL) 1.0. The sourcecode is available on Github

Goals Overview

General information about the goals.

  • virgo:deploy deploys a bundle to a running Virgo instance.
  • virgo:undeploy undeploys a bundle from a running Virgo instance.
  • virgo:refresh refreshs a module deployed in a running Virgo instance.
  • virgo:refreshBundle refreshs a bundle deployed in a running Virgo instance.
  • virgo:start Starts a Virgo instance.
  • virgo:shutdown Stops a running Virgo instance.
  • virgo:immediateShutdown Stops a running Virgo instance immediately.

Usage

Create a Maven project that is using either the Maven bundle plugin or Eclipse Virgo Bundlor to create OSGi bundles. Next add the Virgo Maven plugin to the project pom file. Then it depends on what should be done next.

  • In case of the deploy goal build the artifact first (package)
  • In case of the undeploy goal ensure that a bundle with the given symbolic name and version is available within the Virgo instance
  • In case of the refresh goal ensure that a module has been deployed from the exact same uri before
  • In case of the refreshBundle goal ensure that a bundle with the given symbolic name and version is available within the Virgo instance
