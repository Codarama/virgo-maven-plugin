virgo:shutdown

Full name:

net.flybyte.virgo:virgo-maven-plugin:1.0.0:shutdown

Description:

Stops a running Eclipse Virgo instance. Check the Virgo sourcecode for more information.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
virgoRoot File - The root directory of the Virgo installation.

Optional Parameters

Name Type Since Description
password String - The password used to connect to Virgo via JMX.
Default value is: springsource.
serviceUrl String - The service url of the JMX management server.
Default value is: service:jmx:rmi://localhost:9875/jndi/rmi://localhost:9875/jmxrmi.
truststoreLocation File - The location of the truststore used by Virgo. Instead of definining this property in the pom file it is also possible to use a VM argument -Djavax.net.ssl.trustStore when executing the Maven goal. In case this property has neither been set via VM arguments nor in the POM configuration the virgo root directory will be used and a relative path /config/keystore will be added to it.
user String - The username used to connect to Virgo via JMX.
Default value is: admin.

Parameter Details

password:

The password used to connect to Virgo via JMX.
  • Type: java.lang.String
  • Required: No
  • Default: springsource

serviceUrl:

The service url of the JMX management server.
  • Type: java.lang.String
  • Required: No
  • Default: service:jmx:rmi://localhost:9875/jndi/rmi://localhost:9875/jmxrmi

truststoreLocation:

The location of the truststore used by Virgo. Instead of definining this property in the pom file it is also possible to use a VM argument -Djavax.net.ssl.trustStore when executing the Maven goal. In case this property has neither been set via VM arguments nor in the POM configuration the virgo root directory will be used and a relative path /config/keystore will be added to it.
  • Type: java.io.File
  • Required: No
  • Expression: ${javax.net.ssl.trustStore}

user:

The username used to connect to Virgo via JMX.
  • Type: java.lang.String
  • Required: No
  • Default: admin

virgoRoot:

The root directory of the Virgo installation.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${virgoRoot}
