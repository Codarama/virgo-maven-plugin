virgo:start

Full name:

net.flybyte.virgo:virgo-maven-plugin:1.0.0:start

Description:

Starts an Eclipse Virgo instance by executing the provided startup script.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
virgoRoot File - The root directory of the Virgo installation.

Parameter Details

virgoRoot:

The root directory of the Virgo installation.
  • Type: java.io.File
  • Required: Yes
  • Expression: ${virgoRoot}
